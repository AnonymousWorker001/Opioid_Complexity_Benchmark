# Notebooks

Run notebooks from the repository root or use the shell scripts in `scripts/`.

- `00_preprocess_ctn0094.ipynb`: rebuilds `data/processed/static_timeSeries_new.csv` from CTN-0094 source `.rda` files plus the Excel source file.
- `01_main_benchmark_analysis.ipynb`: main benchmark analysis, models, metrics, cohort-composition stress tests, and figures.
- `02_prefix_cis_figure_s4.ipynb`: supplementary prefix-/sample-level CIS comparison for Figure S4.
