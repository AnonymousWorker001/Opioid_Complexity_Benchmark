# Data folder

`processed/static_timeSeries_new.csv` is included so the benchmark notebooks can run directly.

`external/CTN94WKUDS_E_0330_0820.xlsx` is the additional Excel source file used in preprocessing.

`source/public.ctn0094data-main/data/` is intentionally empty in this repository. To rerun preprocessing from the public CTN-0094 source release, place the `.rda` files (in the `data/` folder) from `public.ctn0094data` and the accompanying `public.ctn0094extra` release in that folder.
